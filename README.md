# Bruce's PPTX Generator

一个专为 AI 编程 Agent 设计的 PPT 生成 Skill，让 Claude Code 能够根据你的需求，从零开始用代码生成专业级 PowerPoint 演示文稿。

## 能做什么

**从零生成完整 PPT**，包含封面、目录、章节分隔页、内容页、总结页，一次性输出可直接使用的 `.pptx` 文件。

**内容页支持多种图形布局**，根据内容逻辑自动选择：

| 内容关系 | 图形布局 |
| -------- | -------- |
| 并列 / 对等的多个要点 | 并列卡片（G1） |
| 一个核心概念 + 多个子项 | 总分布局（G2） |
| 有顺序的步骤 / 流程 | 流程图（G3） |
| 循环 / 相互关联的要素 | 循环图（G4） |
| 有层级 / 优先级的内容 | 金字塔（G5） |

**内置企业科技蓝风格**，开箱即用：深海军蓝 + 亮蓝 + 橙色点缀，适合 AI、安全、产品、解决方案等面向政企客户的正式商业汇报。

**也支持自定义风格**，内置 18 套配色方案（科技、商务、自然、教育、奢华等），4 种设计风格（Sharp / Soft / Rounded / Pill），自由组合。

## 输出风格示例

生成的 PPT 具备以下特点：

- 客户汇报型排版：完整句子、信息密度高，看完幻灯片不需要讲解也能理解
- 每页有独特布局，避免重复，保持视觉节奏
- 所有图形、色块、形状均由代码精确绘制，无需手动调整
- 页码徽章、标题强调线等细节自动生成

## 安装

向你的 AI 编程 Agent 发送以下提示词：

```
帮我安装这个skill：https://github.com/brucevanfdm/bruce-pptx-generator
```

安装完成后，只需告诉 Agent 你要做什么 PPT，它会自动接管。

## 使用

直接描述你的需求即可，例如：

- "帮我做一个介绍公司 AI 安全产品的 PPT，面向金融客户，10 页左右"
- "做一个项目汇报 PPT，包含背景、进展、风险、下一步计划"
- "用企业科技蓝风格，做一个云计算解决方案的演示文稿"

## 依赖

```shell
npm install -g pptxgenjs
```
